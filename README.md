# CLC Billing Automation

Browser-based tooling to support CLC billing and payment workflows.

## Scope

This project is focused on automating CLC billing operations, including:
- Billing and payment workflow support
- Data validation and reconciliation support
- Invoice preparation support
- Reporting support

Potential future scope includes QuickBooks Online automation, while keeping shared billing logic reusable across modules.

## Tech Stack

- JavaScript
- Vite

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start local development:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Notes

- Keep changes focused on CLC billing automation needs.
- Prefer small, safe updates over broad refactors.
